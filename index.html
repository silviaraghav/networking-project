Understanding Computer Networking

What is DNS?

DNS (Domain Name System) translates domain names like www.example.com into IP addresses like 192.168.0.1.

IP Address:

IP Addressing

IP addresses are unique identifiers for devices on a network. There are two types: IPv4 and IPv6.

IPv4: 192.168.0.1

IPv6: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

Protocols

Protocols such as HTTP, TCP/IP, and UDP define rules for how data is transmitted across networks.
